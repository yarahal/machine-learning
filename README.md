# Machine Learning Algorithms

Simple Python implementations of some machine learning algorithms.
Currently includes:

- Linear Regression
- Nearest Neighbors
- Logistic Regression
- Support Vector Machines
- Principal Components Analysis
- Decision Trees
- Random Forests
- Gradient Boosting
- K-Means Clustering
- Neural Networks

## Requirements

- numpy==1.21.0
- cvxopt==1.2.6
