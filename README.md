# Machine Learning Algorithms

Simple Python implementations of some machine learning algorithms.
Currently includes:

- Linear Regression
- Nearest Neighbor Regression
- Logistic Regression
- Nearest Neighbor Classification
- Support Vector Machines

## Requirements

- numpy==1.21.0
- cvxopt==1.2.6
